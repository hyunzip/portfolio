  • START-UP OFFICE
    스타트업 사무실(101호~110호)
    유쾌한 창업으로 글로벌한 콘텐츠의 시작까지!
    스타트 기업들의 사무실을 만들 수 있도록 마련된 공간입니다.
    done 규모 : 총 10개실
    2인실 (전용면적 17㎡, 5평)×4개실 (101호~104호)
    3인실 (전용면적 24㎡, 7평)×6개실 (105호~110호)
    입주자격
    1. 사업화나 런칭을 목표로 콘텐츠 창작 프로젝트를 준비
    또는 진행     중인 창작자(팀) 혹은 스타트업
    2. 입주신청일 기준 기업의 경우
    사업자등록증상 주소, 팀이나 개인의 경우 대표자의 주소가 충북일 것
    3. 창작자(팀) 혹은 창업 3년 미만의 스타트업으로 1명 이상 상근할 것
    4. 콘텐츠산업에 한해 가능
    (만화, 음악, 게임, 영화, 애니메이션, 방송, 캐릭터, 지식정보, 콘텐츠솔루션)

    국세·지방세 체납자,
    금융기관 불량거래자로 제재중인 자, 휴업중인 자
    입주기간
    1년씩 1회 연장 가능 (최장 2년 가능), 1년 단위로 입주 공모 실시
    입주기간 연장을 위해서는 충북CKL의 요청자료를 제출
    schedule 공간 이용시간
    원칙적으로 24시간 사용 가능
